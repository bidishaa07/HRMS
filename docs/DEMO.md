# Hackathon demo guide

1. Open the dashboard and frame the executive brief, workforce health, and live attendance.
2. Click **Check in** to demonstrate the employee flow.
3. Ask Aurora “Who is absent today?” and point out agent routing, tools, and explainability.
4. Ask “Show pending leave”; explain conflict and balance checks, then approve a request in Swagger or the leave module.
5. Ask “Explain June payroll” to show a private, local salary explanation.
6. Show the employee directory and health scores, then ask “Check burnout risk.”
7. Open the Agent Activity panel and explain the ten-agent event-driven architecture.
8. Show Swagger, the database diagram, Mailpit, and the Docker stack to establish production depth.

The full narrative takes 5–7 minutes and does not depend on internet connectivity after images and the Ollama model are pulled.
